Paysera / Mokejimai.lt Payment Method for Event Espresso 4.x.x

== Description ==

<a href="https://www.paysera.com/index.html">Paysera/Mokejimai.lt</a> Payment Method for <a href="http://eventespresso.com/">Event Espresso</a> 4.x.x. Before installation you need to get Paysera Unique project number and Paysera sign password.

== Installation ==

You can either install it automatically from the WordPress admin, or do it manually:

1. Unzip the archive and put the `espresso-paysera-payment` folder into your plugins folder (/wp-content/plugins/).
1. Activate the plugin from the Plugins menu.

After activation go to Event Espresso / Payment Methods there will appear new tab "Paysera". Please add your project id and sign password. For test purposes you can use "Use Paysera test mode"

== Frequently Asked Questions ==

= What is supported versions of Event Espresso =

4.6.x

= What is version of Paysera API =

1.6

= Where I can find Paysera sign password? =

Project password, which can be found by logging in to Paysera system using your user data, selecting “Service management” and “General settings” by a specific project.

= How to turn on Paysera test payments =

For test payments, it is necessary to activate the test mode for a particular project upon logging in: “Service Management” -> “Collection of Payments” (for a specific project) -> “Allow Test Payments” (check).
